<h2>Safety Tips While Using Power Tools</h2>

<div class="alert alert-danger">

Always follow these safety practices before operating any power tool.

<ul class="mt-2 mb-0">

<li>Wear safety goggles or a face shield.</li>

<li>Use gloves suitable for the application.</li>

<li>Read the manufacturer's operating instructions.</li>

<li>Inspect cables and plugs before use.</li>

<li>Never remove safety guards.</li>

<li>Use only the correct accessories for each machine.</li>

<li>Disconnect power before changing blades or discs.</li>

<li>Keep the work area clean and well lit.</li>

</ul>

</div>
